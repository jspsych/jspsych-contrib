# plugin-survey-likert-sd

## Overview

Extention of the survey-likert plugin to enable the use of the SD method.

## Loading

### In browser

```html
<script src="https://unpkg.com/@jspsych-contrib/plugin-survey-likert-sd">
```

### Via NPM

```
npm install @jspsych-contrib/plugin-survey-likert-sd
```

```js
import jsPsychSurveyLikertSd from "plugin-survey-likert-sd";
```

## Compatibility

`@jspsych-contrib/plugin-survey-likert-sd` requires jsPsych v8.0.0 or later.
