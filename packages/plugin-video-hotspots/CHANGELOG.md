# @jspsych-contrib/plugin-video-hotspots

## 1.1.0

### Minor Changes

- [#201](https://github.com/jspsych/jspsych-contrib/pull/201) [`06c648a732a680e4c13d59ee85764d98cb929961`](https://github.com/jspsych/jspsych-contrib/commit/06c648a732a680e4c13d59ee85764d98cb929961) Thanks [@becky-gilbert](https://github.com/becky-gilbert)! - Adds an optional `prompt` parameter to the `image-hotspots` and `video-hotspots` plugins, which is an HTML-formatted string displayed below the stimulus. This can be used to remind participants how to respond. By default, the `video-hotspots` plugin will display the prompt after the video ends, but this can be changed by setting the `show_prompt_on_video_end` parameter to `false`.
