# plugin-plugin-tower

## Overview

A jsPsych plugin for Tower of London/Hanoi style puzzle tasks. Participants move colored balls between pegs to match a goal configuration.

## Loading

### In browser

```html
<script src="https://unpkg.com/@jspsych-contrib/plugin-plugin-tower">
```

### Via NPM

```
npm install @jspsych-contrib/plugin-plugin-tower
```

```js
import jsPsychPluginTower from "plugin-plugin-tower";
```

## Compatibility

`@jspsych-contrib/plugin-plugin-tower` requires jsPsych v8.0.0 or later.
