# plugin-numpad

## Overview

An interactable and responsive numpad supporting both touch and keyboard interaction for numerical input.

## Loading

### In browser

```html
<script src="https://unpkg.com/@jspsych-contrib/plugin-numpad">
```

### Via NPM

```
npm install @jspsych-contrib/plugin-numpad
```

```js
import jsPsychNumpad from "plugin-numpad";
```

## Compatibility

`@jspsych-contrib/plugin-numpad` requires jsPsych v8.0.0 or later.
