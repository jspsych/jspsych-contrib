# extension-tts

## Overview

this extension enables jsPsych trials to have be spoken via Web Speech API's SpeechSynteSynthesis interfaces.

## Loading

### In browser

```html
<script src="https://unpkg.com/@jspsych-contrib/extension-tts">
```

### Via NPM

```
npm install @jspsych-contrib/extension-tts
```

```js
import jsPsychExtensionTts from "extension-tts";
```

## Compatibility

`@jspsych-contrib/extension-tts` requires jsPsych v8.0.0 or later.
