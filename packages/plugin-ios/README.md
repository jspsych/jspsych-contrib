# Inclusion of Other in the Self (IOS)

## Overview

This plugin allows for continuous responses on the Inclusion of Other in the Self (IOS) scale.

## Loading

```js
<script src="https://unpkg.com/@jspsych-contrib/plugin-ios@1.0.0">
```

## Compatibility

jsPsych v7.0
