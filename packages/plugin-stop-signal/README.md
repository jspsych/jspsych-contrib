# plugin-stop-signal

## Overview

This plugin supports the stop signal timeline. It shows a series of 1-2 images and collects user response with buttons. We recommend you use the timeline directly instead of this plugin. 

## Loading

### In browser

```js
<script src="https://unpkg.com/@jspsych-contrib/plugin-stop-signal@1.0.0"></script>
```

### Via NPM

```
npm install @jspsych-contrib/plugin-stop-signal
```

```js
import jsPsychStopSignal from '@jspsych-contrib/plugin-stop-signal';
```

## Compatibility

jsPsych 8.0.0
