# survey-grid

## Overview

Displays a set of questions with Likert scale responses on a grid. Participants respond by selecting a radio button. Includes passive detection of careless/insufficient effort responding via honeypot questions, response timing, straightlining, and zigzagging metrics.

## Loading

### In browser

```js
<script src="https://unpkg.com/@jspsych-contrib/plugin-survey-grid@0.0.1"></script>
```

### Via NPM

```
npm install @jspsych-contrib/plugin-survey-grid
```

```js
import jsPsychSurveyGrid from '@jspsych-contrib/plugin-survey-grid';
```

## Compatibility

jsPsych v8.0.
