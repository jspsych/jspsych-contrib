# plugin-pursuit-rotor

## Overview

Motor tracking task requiring continuous tracking of a moving target with mouse or touch input

## Loading

### In browser

```html
<script src="https://unpkg.com/@jspsych-contrib/plugin-pursuit-rotor">
```

### Via NPM

```
npm install @jspsych-contrib/plugin-pursuit-rotor
```

```js
import jsPsychPursuitRotor from "plugin-pursuit-rotor";
```

## Compatibility

`@jspsych-contrib/plugin-pursuit-rotor` requires jsPsych v8.0.0 or later.
